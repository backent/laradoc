# Laravel with docker

## Installation
Install composer
```bash
./composer install
```
Build containers using docker-compose
```bash
docker-compose up -d
```

Set Project permission
```bash
./set-permission
```
Now you can play with it ^^

## docker command helper

### Run Composer
Execute
```bash
./composer {argument}
```
Example
```bash
./composer install
```

### Run php
```bash
./php {argument}
```
Example
```bash
./php -v
```
